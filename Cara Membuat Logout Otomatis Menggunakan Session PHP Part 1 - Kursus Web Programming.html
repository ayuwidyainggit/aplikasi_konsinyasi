Cara Membuat Logout Otomatis Menggunakan Session PHP Part 1

by Irwan Deswitansyah | Jan 11, 2017 | MySQL, PHP, Tips dan Trik, Website | 0 comments

Halo teman-teman pada tutorial kali ini saya akan menjelaskan tentang cara membuat  Cara Membuat Logout Otomatis Menggunakan Session PHP Part 1, untuk Membuat Logout Otomatis Menggunakan Session PHP ada beberapa tahap yang akan kita buat yaitu :

  1. Membuat database
  2. Membuat koneksi, yang berfungsi untuk menghubungkan file yang telah kita buat supaya dapat terkoneksi dengan database
  3. Membuat file index.php, yaitu berfungsi sebagai halaman yang pertama yang akan di akses oleh user
  4. Membuat proses, yaitu berfungsi untuk proses pengecekan, apakah username yang kita masukan telah benar atau tidak
  5. Membuat halaman home, jadi ketika username atau password yang dimasukan telah benar maka akan masuk kehalaman home, dan pada nantinya pada halaman home inilah Logout Otomatis berfungsi, dengan waktu yang telah kita tentukan nantinya.
  6. Yang terakhir kita akan membuat proses logout nya.

Namun pada tutorial Cara Membuat Logout Otomatis Menggunakan Session PHP Part 1 kali ini kita akan membahas terlebih dahulu yaitu membuat database dan membuat file koneksinya dan untuk proses selanjutnya akan saya bahas pada tutorial selanjutnya.

Oke teman-teman kita langsung saja masuk pada langkah yang pertama yaitu kita akan membuat databasenya terlebih dahulu, dan silahkan teman-teman buka localhost/phpmyadmin nya, selanjutnya kita akan membuat database baru dengan nama database yaitu, login selanjutnya pada database login silahkan teman-teman buat table baru dengan nama tabel user lebih kurang struktur table yang akan kita buat nantinya seperti dibawah ini,.

Nah jika database dan tabelnya telah kita buat berikutnya kita akan membuat file koneksinya, dan silahkan teman-teman simpan script dibawah ini dengan nama koneksi.php
